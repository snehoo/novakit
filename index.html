Do I need a paid Claude account? +
No. The free Claude plan is enough to get started and see the skill working. A Pro plan unlocks longer outputs and is worth it if you're generating longer documents like business plans or full pitch decks.
How do I install a skill? +
Download the .skill file. Open Claude, click Customize in the left sidebar, go to Skills → + → Create skill → Upload a skill. Drag and drop your file. Done. It appears in your slash-command list immediately and works on both claude.ai and the Claude desktop app.
What makes these different from regular prompts? +
A regular prompt is a static instruction. NovaKit skills check live trends before every run, ask you 3 narrowing questions to avoid generic output, run a quality gate that rewrites weak drafts before you see them, and self-improve over time. Run the same skill today and in six months — you'll get different results because what works will have changed, and the skill will know.
