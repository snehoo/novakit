Composition Brief · Layout + Face + Text
Every design decision, justified by CTR data.
LAYOUT
Subject position: Left two-thirds
Text zone: Top-right safe area
Background: Dark gradient — dark-mode optimised
Focal points: 1 (face only) — no competing elements
Eye flow: Face → text overlay → edge

FACE
Expression: Focused concern — raised inner brows, not shock
Gaze: Direct to camera
Frame coverage: ~35% of total width

COLOUR PALETTE
Primary: #0a0a0f · Accent: #00e5ff · Text: #ffffff
Dark backgrounds with neon accents outperform bright thumbnails on YouTube's grey/white UI — especially for tech and dev niches.
Do / Don't List · Niche-specific
Calibrated to your niche. Not a generic checklist.
DO:
✓ Use real skin texture — audiences skip hyper-polished AI faces
✓ Keep text to 3 words max — legible at 120px wide on mobile
✓ Let gaze point toward the text overlay
✓ Test a second variant: same composition, swap dark bg for light

DON'T:
✗ Use the classic wide-open shock face — reads as low-effort in 2026
✗ Place any text or face in the bottom-right (YouTube timestamp)
✗ Add a second focal element — one hero element beats two every time
✗ Over-promise in the text overlay — algorithm penalises low watch time
Do/don't rules are calibrated to the specific niche, emotion, and CTR trigger combination — not copy-pasted universals.
Designer / Editor Notes
Model routing, A/B guidance, algorithm alignment.
AI Model: Flux 2 Pro for the face render; Midjourney V7 for background cinematic grade if compositing separately.

CTR Psychology: Curiosity gap — "show the what, hide the how" — performs for tutorial-adjacent content. Paired with direct gaze and a single warning-tone text overlay, this combination lifts initial CTR 30–50% vs descriptive titles.

Algorithm Note: Keep thumbnail-content alignment honest. High initial CTR with low watch time signals clickbait and suppresses reach over 7–14 days.

A/B Test: Same composition, swap cyan accent for amber — measure 48-hour CTR difference.
